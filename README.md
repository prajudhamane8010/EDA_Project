# 🌸 MLOps Laboratory - Experiment 2

## 📌 Project Title
Exploratory Data Analysis (EDA) Automation using Python

## 🎯 Aim
To perform Exploratory Data Analysis (EDA) on the Iris dataset using Python and generate useful insights such as summary statistics, missing values, dataset information, and correlation matrix.

## 🛠️ Software Required
- Python 3.10+
- VS Code
- Pandas
- Matplotlib
- Seaborn

## 📂 Project Structure

```
EDA_Project/
│
├── data/
│   └── iris.csv
│
├── eda.py
└── README.md
```

## 📚 Libraries Used

```bash
pip install pandas matplotlib seaborn
```

## ▶️ How to Run

```bash
python eda.py
```

## 🔍 Procedure

1. Create the project folder.
2. Create a `data` folder.
3. Place `iris.csv` inside the `data` folder.
4. Install the required libraries.
5. Create `eda.py`.
6. Load the dataset using Pandas.
7. Display the first five records.
8. Display dataset information.
9. Display summary statistics.
10. Check for missing values.
11. Generate the correlation matrix.
12. Run the program and observe the output.

## 📊 Output

The program displays:

- First five records of the dataset
- Dataset information
- Summary statistics
- Missing values
- Correlation matrix

## ✅ Result

The Iris dataset was loaded successfully. Exploratory Data Analysis (EDA) was performed, and the first five records, dataset information, summary statistics, missing values, and correlation matrix were generated successfully.
